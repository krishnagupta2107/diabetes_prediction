# Diabetes Prediction System

Machine Learning web application for predicting diabetes risk using the PIMA Indians dataset.

## Features
- Logistic Regression ML model with 78.5% accuracy
- Flask web application
- Real-time predictions
- Beautiful responsive UI

## Local Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Train model: `cd model && python train_model.py`
3. Run app: `python app.py`
4. Open: http://localhost:5000

## Deployment
This app is ready to deploy on Render, Heroku, or PythonAnywhere.

### Render Deployment:
1. Push to GitHub
2. Connect to Render
3. Auto-deploy!

## Model Performance
- Accuracy: 78.5%
- Precision: 75.2%
- Recall: 71.8%
- F1-Score: 73.4%
